How Webpages Are Built

HTML Syntax
HTML stands for Hyper Text Markup Language, it is the language that webpages are created in and the first leg in the front end web development trio. HTML documents are made up of elements which are represented by tags. Almost all elements have an opening and a closing tag. The below example is a minimum HTML document. 
                    
                      <html>
                        <head>
                          <title>My First Webpage</title>
                        </head>
                        <body>
                          This is some body text
                        </body>
                      </html>
